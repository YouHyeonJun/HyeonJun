# Java

두 정수 a, b 사이에 속한 모든 정수의 합을 구하기

제한 조건  
a와 b는 음수가 올 수 있다  
a와 b의 대소관계는 정해져있지 않다.


```java
class Sum {
  public long sum(int a, int b) {
      long sum = 0;
      int temp = 0;
      if (b<a){
          temp = b;
          b = a;
          a = temp;
      }
      
      for(int i=a;i<=b;i++){
          sum += a;
      }      
      return sum;
  }
}
```
